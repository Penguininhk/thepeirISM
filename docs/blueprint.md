# **App Name**: The PIER - Harbour School Portal

## Core Features:

- Announcements Board: Display school-wide and class-specific announcements.
- Classwork Submission: Enable students to submit assignments and view grades.
- Attendance Tracking: Allow teachers to mark attendance and students to view their attendance record.
- Course Selection: Enable students to select courses for upcoming semesters.
- User Authentication: Secure user accounts with role-based permissions.
- Personalized Dashboard: Display relevant information (announcements, upcoming assignments, etc.) based on the user's role and classes.
- Student Login: Dedicated login page for students in Hong Kong
- Teacher Login: Dedicated login page for teachers in Hong Kong

## Style Guidelines:

- Primary color: Deep blue (#293B7F), reflecting the harbor and a sense of trustworthiness and stability.
- Background color: Light blue (#DDE6ED), offering a calm and spacious feel.
- Accent color: Soft orange (#D9534F), used to draw attention to calls to action, announcements, or important information.
- Body and headline font: 'PT Sans' for a modern, readable style suitable for both headings and body text.
- Use simple, modern icons from a consistent set, relating to navigation items and common actions.
- A clean, card-based layout will present information in an organized way.
- Subtle animations provide feedback on user actions and smooth transitions between pages.
